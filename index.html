Documentation
Input
Note:

The input must be valid!

The valid output is Positive Numbers

Here is some example that is forbidden:

\[\text{e.g.}: \{ 0, -1, -2.5, 1000000000000001 \} \]

Here is the requirement of the output with some examples:

\[\text{Output} = \{x \mid 0 \lt x \le 10^{15} \mid x \in \mathbb{R} \}\]

\[\text{e.g.}: \{ 2, 1.618, 1024, 0.25 \} \]

If you input the forbidden numbers, the calculator will give you a warning and suggest you to change your input into the valid input
Output
Note:

If you have enough variables information,

The output will automatically shown

Normally, you just need to input all things you need

and left one of the variable with a NaN (null) value

Example of the I/O on the area section:

Input: Base(b) variable

Input: Area(A) variable

Output: Height(h) variable

Else if you input all of the variable, then:

The output would be boolean

e.g. true, false, or "The triangle doesn't exist"
